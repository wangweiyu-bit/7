🚀 快速启动后端

📋 启动步骤

  1. 安装 Node.js - 从 nodejs.org 下载安装
  2. 安装依赖 - 在 backend 目录运行: npm install
  3. 启动服务 - 运行: node src/server.js
  4. 测试连接 - 点击下面的"检查服务状态"按钮
💡 提示: 如果后端未运行，所有测试按钮都会显示错误信息。请先按照上述步骤启动后端服务。

🔐 认证相关 API

POST /api/auth/register
用户注册（需要管理员审核）
POST /api/auth/login
用户登录（邮箱/密码）
GET /api/auth/me
获取当前用户信息
GET /api/auth/wechat/start
微信 OAuth 登录开始
GET /api/auth/wechat/callback
微信 OAuth 回调处理

👑 管理员 API

GET /api/admin/users
获取用户列表（支持筛选）
POST /api/admin/users/:id/approve
审核通过用户

💬 匿名互动 API

GET /api/anon/posts
获取匿名帖子列表（分页、搜索）
POST /api/anon/posts
发布匿名帖子
POST /api/anon/posts/:id/like
点赞帖子
GET /api/anon/posts/:id/comments
获取帖子评论
POST /api/anon/posts/:id/comments
添加评论

📁 文件上传 API

POST /api/upload
上传文件（支持多文件，最大4个）

📅 预约系统 API

GET /api/bookings
获取用户预约记录
POST /api/bookings
创建预约（book/queue/cancel/checkin）

🔗 功能对应关系

  • 会员预约 → /api/bookings (POST/GET)
  • 匿名发帖 → /api/anon/posts (POST/GET)
  • 匿名评论 → /api/anon/posts/:id/comments (POST/GET)
  • 匿名点赞 → /api/anon/posts/:id/like (POST)
  • 文件上传 → /api/upload (POST)
  • 用户登录 → /api/auth/login (POST)
  • 微信登录 → /api/auth/wechat/start (GET)

🗄️ 数据库表结构

users - 用户表
id, email, password, name, role, approved, wechat_openid, session_version, active_device_id, created_at
oauth_states - OAuth 状态表
state, created_at
bookings - 预约表
id, user_id, date, slot_id, name, phone, action, status, created_at
anon_posts - 匿名帖子表
id, board, device_id, title, content, tags, media, likes_count, comments_count, status, created_at
anon_comments - 匿名评论表
id, post_id, device_id, content, created_at
anon_likes - 匿名点赞表
id, post_id, device_id, created_at
